# Chess ChatBot with Ollama

A simple chess-focused chatbot built with **Streamlit** and **Ollama (phi3:mini)**.  
It acts like a tutor: answering questions about chess rules, openings, strategies, and history in a fast, ChatGPT-style interface.

---

## 🚀 Features
- Chat-style interface (like ChatGPT).
- Runs locally with Ollama (no internet required after setup).
- Focused on chess knowledge (rules, openings, tactics, famous players).
- Lightweight and fast.

---

## 📦 Requirements
- Python 3.9+
- [Ollama](https://ollama.com) installed on your system
- `phi3:mini` model pulled locally:
  ```bash
  ollama pull phi3:mini
